# Web Security & OWASP Top 10
This repository contains practical notes and examples related to the OWASP Top 10.

The goal of this project is to provide a simple and hands-on way to learn about common vulnerabilities, how they happen, and why they matter.

---

## OWASP Top 10

The OWASP Top 10 is a globally recognized list of the most important security risks in modern web applications.  
Examples of common risks include:

- Broken Access Control  
- Injection Attacks  
- Cross-Site Scripting (XSS)  
- Security Misconfiguration  
- Sensitive Data Exposure  

More info: https://owasp.org/www-project-top-ten/

---

## Repository Structure

Each folder in this repository focuses on one OWASP Top 10 category.

Example:

- **A01-broken-access-control/**  
  Demonstrates how unauthorized users can bypass restrictions.

Each section typically includes:

- A short explanation of the vulnerability (both in Turkish and English)
- Example project containg the risks
- Example secure version of that project
